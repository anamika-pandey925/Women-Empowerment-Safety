# Suraksha: Women Safety & Empowerment Platform 🛡️

Suraksha is a comprehensive, multi-layered safety ecosystem designed to empower women through technology, community, and education. It integrates real-time safety tools, anonymous reporting, and career-building resources into a single, intuitive platform.

## 🌟 Key Features

### 🔒 Safety & Emergency
- **SOS Button with Live Location**: Instant alerts to trusted contacts and emergency services.
- **Fake Call**: Simulation of incoming calls to safely exit uncomfortable situations.
- **Safe Route Finder**: Integrated Google Maps to locate nearby police stations and safe zones.
- **Crisis Support**: 24/7 hotline and legal advocacy resources.

### 📹 Video Reporting
- **Incident Capture**: Upload or record safety concerns directly to the safety department.
- **Secure Handling**: Encrypted submission for review and immediate action.

### 💬 Community & Empowerment
- **Anonymous Reporting**: Report unsafe zones or incidents without revealing identity.
- **Community Forum**: A safe space for sharing experiences, seeking advice, and support.
- **Know Your Rights**: Detailed education on POSH, Domestic Violence Act, and Cyber Laws.

## 🎨 Integrated Projects (Portfolio)

### 📊 TaskFlow
A high-performance **MERN Stack** task management platform.
- **Stack**: React, Next.js, Node.js, MongoDB, JWT Authentication.
- **Features**: Real-time dashboard, task CRUD operations, priority filtering, and secure profile management.

### 🌱 IoT Precision Agriculture
AI-driven soil monitoring and crop recommendation system.
- **Hardware**: Arduino, Soil Moisture & pH Sensors.
- **Software**: Python Analytics, Data Visualization, AI Recommendation Engine.

## 🛠️ Technical Stack

- **Frontend**: React, Next.js, TypeScript, TailwindCSS, GSAP (Animations), Lucide-React.
- **Backend**: Node.js, Express.js, MongoDB, Mongoose, JWT.
- **Translations**: Full support for **English** and **Hindi (हिन्दी)**.
- **Deployment**: Netlify (Frontend), Railway (Backend).

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Run Locally**:
   ```bash
   npm run dev
   ```
3. **Environment**: Ensure `.env` is configured with necessary API keys and MongoDB URI.
